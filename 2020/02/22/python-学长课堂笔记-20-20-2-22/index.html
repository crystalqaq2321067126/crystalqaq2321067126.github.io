##函数
定义：函数是组织好的，可重复使用的，用来实现单一，或相关联功能的代码段。
函数可以提高代码的重复利用率，前面学习到了内置函数，现在我们可以自定义函数。
##函数的定义
1.函数代码块以 def 开头，后接函数标名和括号 () 最后就是一个英文冒号 :
2.传递的参数必须放在括号里面，圆括号内定义参数
3.函数内容必须缩进
4.函数的第一行使用文档字符串形式存放函数说明
